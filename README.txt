== Installation ==

Method 1:
<ol>
<li> Download the plugin sort-products-in-cart-order-lite-for-woocommerce.zip </li>
<li> Unpack the sort-products-in-cart-order-lite-for-woocommerce.zip file and extract the sort-products-in-cart-order-lite-for-woocommerce folder </li>
<li> Upload the plugin folder to your /wp-content/plugins/ directory </li>
<li> Go to WordPress dashboard, click on Plugins from the menu </li>
<li> Locate the Sort WooCommerce Products in Cart and Order Lite and click on Activate link to activate the plugin. </li>
</ol>

Method 2:
<ol>
<li> Download the plugin sort-products-in-cart-order-lite-for-woocommerce.zip </li>
<li> Go to WordPress Dashboard-Plugins-Add New </li>
<li> Click on Upload Plugin link from top </li>
<li> Upload the downloaded sort-products-in-cart-order-lite-for-woocommerce.zip file and click on Install Now </li>
<li> After installation, click on Activate Plugin link to activate the plugin. </li>
</ol>
